# 4390-Capstone-project
Group of 4 developers come together to redesign the car website: https://www.lingscars.com/ that has several usability issues.
Our goal is to redesign the website  improving it's usability and visual layout.
Getting Started 
To run the application locally:
Open index.html in your project folder
Right click inside the file and select open live server. 
Navigate thrugh the site and explore all the pages.
